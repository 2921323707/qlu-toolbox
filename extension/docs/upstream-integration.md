# 与 GitHub 主版本的整合说明

整合基线：`2921323707/qlu-toolbox` `main`，检查时提交为 `d59786d`（桌面版 1.1.0）。

## 已整合

- 品牌：扩展图标直接使用主版本 `assets/qlu-toolbox.png` 的透明资产，并生成 Manifest 所需的 16、32、48、128 像素版本。
- 成绩导出：浏览器版沿用主版本的成绩页入口、导出接口和九个成绩字段，同时继续使用现有的 ZIP/XLSX 安全解析边界。
- GPA：移植主版本的 5.0 制数字、字母和中文等级换算规则；只把具有有效学分及“总评/总评成绩”的课程纳入加权结果。
- 产品结构：浏览器版只展示已经可用的工具，不呈现未定义或不可操作的占位功能。
- 隐私：继续坚持本地处理；扩展不持久化 Cookie、账号、成绩或查询记录。

## 保持平台独立

- 桌面版的 Electron 外壳、Python Worker、Playwright 浏览器、备用 Chromium、任务数据库和更新器没有复制进扩展。
- 浏览器版仍只声明 `activeTab` 与 `scripting`，不新增常驻站点权限、content script 或扩展存储。
- 两个发行版本共享品牌与业务规则，但版本号独立：浏览器版从 0.2.0 开始记录整合后的迭代。

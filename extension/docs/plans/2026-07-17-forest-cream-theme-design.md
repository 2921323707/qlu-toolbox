# QLU ToolBox 森林绿奶油主题设计

## 视觉方向

新版放弃过于无菌的纯白方案，改用“森林编辑部”方向：深森林绿负责品牌、标题和主操作，奶油白承担大面积阅读背景，鼠尾草绿用于状态和轻量层级，香槟金只强调 GPA、学分与总评。相比酒红奶油，它更贴近学术和校园场景；相比黑金，它在长时间查看成绩时更轻松。配色不包含蓝色，也不使用常见的科技渐变。背景保留非常弱的纸张颗粒与自然光斑，让界面有质感但不抢夺数据注意力。

## 组件规则

弹窗只保留品牌、页面状态、当前工具和隐私说明，不展示未定义的占位功能。页面引导与工具卡使用奶油白表面、森林绿细边和柔和绿灰阴影；主按钮使用深浅森林绿渐变，成功状态使用低饱和鼠尾草绿。成绩浮层沿用相同令牌，顶部指标保持白色卡片，GPA 指标加入香槟边界。课程卡使用淡绿灰分隔线，总评单元格以鼠尾草绿和香槟金的极浅混合底色突出，不改变阅读顺序。

## 可访问性与验证

深森林绿正文与奶油白背景保持清晰对比，按钮使用白字；焦点环改为半透明森林绿。加载、错误、隐藏分数、Escape 关闭、Tab 循环和减弱动效行为不变。验收覆盖非成绩页引导、成绩页就绪、1440×900 宽屏浮层和 430×860 窄屏浮层，要求无横向溢出、无蓝色像素、预留板块完全消失，并通过全部单元测试、Manifest/CSP 检查和确定性打包。

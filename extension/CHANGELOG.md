# 更新日志

## 0.2.3 - 2026-07-17

### 更改

- 删除成绩页跳转卡片、按钮及标签页导航逻辑，改为紧凑的纯提示状态条。
- 不在成绩查询页时仅提示用户手动进入对应页面。

## 0.2.2 - 2026-07-17

### 更改

- 删除弹窗中的三个预留板块及其无用样式。
- 弹窗内容收紧为页面引导、当前工具和隐私说明。
- 视觉主题更新为森林绿、奶油白与香槟金，不使用蓝色。

## 0.2.1 - 2026-07-17

### 更改

- 弹窗与成绩浮层从深色流动玻璃改为高级白色主题。
- 主题彻底移除蓝色，改用暖白、象牙白、雾灰、石墨黑与香槟金。
- 扩展 Logo 与工具栏图标统一转换为透明灰阶版本。
- 功能结构、成绩页引导、GPA 规则、权限与隐私边界保持不变。

## 0.2.0 - 2026-07-17

### 新增

- Liquid Glass 弹窗和页内成绩浮层，支持减弱动效、键盘焦点和窄屏布局。
- 非成绩页状态识别与“一键前往成绩查询页”引导。
- 与桌面版 1.1.0 一致的 5.0 制加权 GPA 估算。
- 学业效率、校园生活、自由扩展三个待定义板块。

### 更改

- 扩展品牌统一为 QLU ToolBox，并采用 GitHub 主版本的透明 Logo。
- 浏览器版版本号更新为 0.2.0，Manifest 展示桌面版整合基线。
- 工具说明、架构文档、隐私边界和使用流程同步更新。

### 保持不变

- 仍只声明 `activeTab` 与 `scripting`。
- 无远程运行时代码、遥测、扩展存储或长期站点权限。
